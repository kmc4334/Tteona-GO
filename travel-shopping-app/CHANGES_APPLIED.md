## 실행 방법

### 옵션 1: 캐시 클리어 후 재시작 (권장)
```bash

```

### 옵션 2: 일반 시작
```bash
npm start
# 또는
npx expo start
```

## 주요 기능

### 📝 일정 작성 탭
1. **기본 정보 입력**
   - 여행 제목
   - 여행지 (예: 제주, 부산, 서울)
   - 출발일/도착일 (달력 팝업)
   - 인원, 교통수단, 메모

2. **AI 추천 옵션**
   - ✅ 체크 시: 성향 분석 기반 자동 일정 생성
   - ❌ 체크 해제 시: 빈 일정표 생성

3. **세부 일정 작성하기 버튼**
   - 클릭 시 일정표 화면으로 전환
   - Day별 시간대별 일정 표시
   - 장소 추가/삭제/수정 가능
   - 저장 버튼으로 라이브러리에 저장

### 📚 라이브러리 탭
- 저장된 여행 일정 목록
- 클릭하여 불러오기
- 삭제 버튼으로 제거

## AI 추천 로직

1. **성향 기반 추천**
   - AsyncStorage에서 `personalityType` 확인
   - personalityTypes.json에서 해당 성향의 해시태그 참조

2. **지역별 장소 데이터**
   - 제주: 성산일출봉, 한라산, 협재해변, 만장굴 등
   - 부산: 해운대, 감천마을, 광안리, 태종대 등
   - 기타: 기본 관광지/맛집/카페 템플릿

3. **시간대별 일정**
   - 09:00 - 출발/도착
   - 10:00 - 오전 관광 (2시간)
   - 12:30 - 점심 (1시간)
   - 14:30 - 오후 활동 (2시간)
   - 17:30 - 석양 감상 (1시간)
   - 19:00 - 저녁 (1.5시간)

## 주의사항

⚠️ **Metro bundler 캐시 문제**
- 화면이 흰색만 표시되거나 오류가 발생하면 `RESTART_APP_CLEAN.bat` 실행
- 캐시를 클리어하지 않으면 이전 ScheduleScreen 참조가 남아있을 수 있음

⚠️ **ScheduleScreen은 제거되지 않음**
- 파일은 그대로 유지되지만 더 이상 사용되지 않음
- 필요시 나중에 삭제 가능
